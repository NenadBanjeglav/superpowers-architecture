# Installation

## Codex

V1 targets Codex.

### Local Development

Clone the repo:

```powershell
git clone https://github.com/NenadBanjeglav/superpowers-architecture.git
```

Install it through your Codex local plugin workflow. If you maintain a personal Codex marketplace, add this plugin to that marketplace and install it from the Codex app.

### Expected Plugin Identity

- Display name: `Superpowers Architecture`
- Package name: `superpowers-architecture`
- Repository: `https://github.com/NenadBanjeglav/superpowers-architecture`

## Claude

Claude support is not shipped in V1. See `docs/claude-roadmap.md`.
